## Tech Stack

**Client:** handlebars, Sass, Webpack

**Server:** Node, Express, docker, docker-compose

**Db:** Mongo

## Installation
Server in nodemo and front in webpack

-[Dev]
1) Install all npm
```bash
npm i
```
2) Run mongo dev
```bash
make i_up
```
3) Run dev nodemon
```bash
npm run dev
```

-[Build]
1) Install all npm
```bash
npm i
```
2) Run build
```bash
npm run build
```

-[docker]
Up docker
```bash
make up
```
